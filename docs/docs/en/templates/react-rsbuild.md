# React + Rsbuild

Preset includes:

- @rsbuild/core + @rsbuild/plugin-react
- UnoCSS, Sass, SVGR
- Auto Import (react, react-router)

## Run

```bash
pnpm install
pnpm dev
```

## Important files

- rsbuild.config.ts: build config
- src/router/index.tsx: router config
- src/layout/default: base layout
